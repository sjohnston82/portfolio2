About Me

Hi! I'm Stephen, a full-stack web developer who loves problem-solving and learning new technologies.

I proudly served in the US Army as an Airborne Infantryman. After serving, I worked for nearly a decade as a food packaging printer.

Last year, I decided to take the leap to do something I've found myself to be very passionate about: coding.

I am a graduate of PDX Code Guild's Python Full-Stack Course as well as their Advanced Javascript and MERN Stack Bootcamp.

When I'm not coding, I love playing PC games, reading sci-fi and fantasy novels, fishing, and spending time with my family.

Some of my Projects

JobTracker

JobTracker is an all-in-one job application information tracker made with the MERN stack.

Features image and document uploads, ability to save all important job info and more. Styled with SASS and hosted on AWS.

Yahtzee!

An online multiplayer version of Yahtzee built using the MERN stack and socket.io. Play full games and chat.

This project was built by myself and another student as a capstone project for PDX Code Guild's AdvJS and MERN Stack Course.

QuarterMaster

QuarterMaster is a pantry inventory manager site. Add items, keep track of expiration dates and make grocery lists.

This project was made as a capstone for PDX Code Guild's Full Stack Boot Camp. It uses Vue and has django for the back end.

Weather App

This weather app uses the openWeather API to find accurate weather and forecasts.

It is built with JavaScript and dynamically changes the background image based on the weather of the location you are searching for.

Web Scraper

The purpose of this program is to scrape relevant data from Amazon product urls in a csv. Inside the csv is also a buy below threshold, which will print an alert as well as send an email notifying the recipient that the price is now below the threshold.

Blackjack

Single player blackjack game made with Vue. It uses the Deck of Cards API to deal cards.
